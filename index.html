📄 Project 1 — Fraud Detection in Automobile Insurance Claims

This project utilizes an insurance claims dataset from Kaggle to detect and forecast possible fraudulent claims. Following data preparation and cleaning, a range of statistical and machine learning methods were employed to categorize claims as fraudulent or non-fraudulent. The analysis helps insurance companies with risk segmentation, fraud detection, and equitable pricing strategies.

Highlights:

  • ✔️ Cleaned and preprocessed insurance claim data
  • ✔️ Used data visualization (Matplotlib, Seaborn) to detect patterns associated with car accidents and fraud
  • ✔️ Applied machine learning (Scikit-learn) models to develop a risk rating model
  • ✔️ Insights help in detecting high-risk customers and preventing fraud

📊 Project 2 — YouTube Data Analysis & Web Scraping

This project centers on web scraping and analyzing YouTube videos and comments associated with the keyword “Snow White”. Using the YouTube Data API, metadata and comments from videos were gathered and examined. The project employed text mining, sentiment analysis, and visualization methods to investigate audience engagement and perspectives.

Highlights:

  • ✔️ Scraped YouTube videos and comments with the keyword “Snow White”
  • ✔️ Processed text using NLTK, TextBlob, and word clouds for insights
  • ✔️ Conducted sentiment analysis to gauge audience response
  • ✔️ Built simple regression models to identify relationships between engagement metrics

🎬 Project 3 — Business Intelligence on Movie Industry Data (Excel & DAX)

This project employs ETL (Extract, Transform, Load) processes and Business Intelligence methods to analyze the financial and performance metrics of over 4,400 movies. Using Excel’s data model, pivot tables, and DAX calculations, it derived multiple insights into seasonal trends, audience ratings, and theater performance. The aim was to transform raw movie datasets into actionable business insights for stakeholders in the entertainment industry.

Highlights:

  • ✔️ Performed ETL operations: data cleaning, type conversion, missing value handling
  • ✔️ Built relational tables and integrated into Excel’s data model
  • ✔️ Conducted analysis with pivot tables + DAX formulas (ROI, profit, revenue)

👥 Project 4 — HR Attrition Analysis with Power BI

This project examines a comprehensive employee attrition dataset with over 59,000 records to uncover factors affecting employee turnover and suggest practical HR strategies. Through ETL procedures, data modeling, DAX measures, and Power BI dashboards, it offers insights into industry sectors, demographic groups, and workplace conditions that drive resignation patterns.

Highlights:

  • ✔️ Performed ETL operations on employee, company, and performance data
  • ✔️ Built relationships between tables
  • ✔️ Created DAX measures and calculated columns
  • ✔️ Designed interactive Power BI dashboards for HR decision-making
